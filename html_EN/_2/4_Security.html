DOCUMENT SECURITY

It prevents the leakage of important information from the outside by applying encryption and authority control from the time of data creation.

MAIN FUNCTION

Document Security Policy Management

Administrator can apply document security policy by document grade and user according to integrated authority policy

Stored according to organizational policy change and organizational change. Various document usage policies can be applied, such as whether editing / printing is possible and the validity period of document usage

Usage control and screen capture prevention

When reading encrypted documents, check the set period of use and control the reading of documents when the period of use is exceeded.

Screen capture control for encrypted documents

Applying pattern recognition capture prevention technology rather than process control method of simple capture program

User permission control

Unauthorized documents are not accessible when viewed inside or outside the company

Copy & Paste from encrypted security document to normal document is impossible

Control the application (menu, etc.) according to the encrypted document setting authority (edit, save, print)

Check on / offline authenticity check

To prevent leakage of documents, print watermarks and printout information together by inserting watermarks when printing documents.

Output log is managed by document security server when printing.

Use history management

Provides usage history inquiry function by sending document usage history such as reading, saving, editing, and output to server

FEATURES AND BENEFITS

Information leak prevention

Encrypt all documents you create to prevent the leakage of sensitive information

Unauthorized access control

Prohibit unauthorized users from using documents by controlling their permissions and preventing access

Distribution Channel Monitoring

Monitoring the distribution path of important information by managing usage history such as creation, output, and external transmission of encrypted documents
